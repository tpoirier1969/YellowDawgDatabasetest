Fish Log Test v2
================

What this build adds
--------------------
- full structured fishing log form
- localStorage persistence
- quick stats
- filtering
- Leaflet map preview
- export / import JSON
- coordinate entry by clicking the map

How to use
----------
1. Open index.html on GitHub Pages or a local server.
2. Fill out the form.
3. Click the map to set coordinates, or leave them blank.
4. Save the entry.
5. Review saved entries, filter them, and jump to mapped locations.

What comes next
---------------
Next build should wire these fields to Supabase and then connect the entries to the actual river map workflow.
