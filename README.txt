Fish Map Test v10.3

What changed
- added a broader Midwest fish species list for logging and filtering
- Add Log now opens the form and tries device GPS first when available
- added Use My Location and Pick on Map controls in the location section
- added an optional Waypoint / Spot Name field for boat spots, reef edges, trolling passes, and similar locations
- saved logs now retain location source info (device GPS / map-picked / adjusted marker)
- marker popup now shows coordinates so spots are easier to identify later

Notes
- device location requires browser permission and usually works only on HTTPS or localhost
- if device location fails or is denied, Pick on Map is still available as the fallback
