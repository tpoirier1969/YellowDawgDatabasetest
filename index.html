Read Me

What this app is, what it tracks, and how sharing works.

About This Project

This app is a working fishing log and trip-analysis tool built to help anglers record conditions, methods, and results in a more useful way than a simple notebook or map pin. It is already in active use, and new features are being added as the system grows.

What It Tracks

You can log trip details, water and weather conditions, fishing style, bait or presentation choices, and results. Over time, that information becomes more valuable because patterns begin to emerge across seasons, waters, species, and techniques.

Sharing Options

This version of the app gives you two sharing controls in Options.

  • Share Name Publicly: choose whether shared logs show your angler name or appear as Anonymous.
  • Location Sharing Depth: choose how specific the shared location text can be.
    • Water Type Only: other people can tell whether the trip was on a river, stream, lake, pond, or Great Lake, but not the named water.
    • Body of Water Name: other people can see the water name, such as Teal Lake or Boardman River.

Shared map placement follows a simple rule: it uses the nearest public access point when one is found. If no access point is found, it uses the saved fishing spot itself. It does not intentionally throw markers into random areas.

Where It’s Going

This is a hobby project under active development. New features will continue to be added, with the long-term goal of building a more predictive, AI-driven fishing experience. As more data is logged, the system will be able to do a better job identifying patterns and help anglers make smarter decisions about where, when, and how to fish.

Development Note
