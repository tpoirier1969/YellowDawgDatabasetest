Read Me

What this app is, what it tracks, and how sharing works.

About This Project

This app is a working fishing log and trip-analysis tool built to help anglers record conditions, methods, and results in a more useful way than a simple notebook or map pin. It is already in active use, and new features are being added as the system grows.

What It Tracks

You can log trip details, water and weather conditions, fishing style, bait or presentation choices, and results. Over time, that information becomes more valuable because patterns begin to emerge across seasons, waters, species, and techniques.

Sharing Levels

This project supports different levels of location sharing so you can decide how precise a spot should appear.

  • Private: keeps the exact location for your own records. Example: if you marked a specific bend in a trout stream or a precise reef edge on Lake Superior, only you would see that exact point.
  • Approximate: shifts the display to a nearby general area instead of the exact spot. Example: instead of showing the exact place where you caught fish on a river, it might show a nearby road crossing, access point, or broader stretch of water.
  • Public / Shared View: shows a generalized location meant for sharing without giving away the exact honey hole. Example: on an inland lake it may show the public boat launch rather than your exact drift line; on a river it may show the access site rather than the exact hole or seam you fished.

The goal is to make the log useful while still protecting the more sensitive details of where fish were actually found.

Where It’s Going

This app is already in production, but it is still evolving. New features will continue to be added, with the long-term goal of building a more predictive, AI-driven fishing experience. As more data is logged, the system will be able to do a better job identifying patterns and helping anglers make smarter decisions about where, when, and how to fish.

Development Note

This is a live project under active development. Improvements and new features are being added over time, and some parts of the app may change as the system grows.
